Hexo博客搭建教程

2022/9/4 教程

我搭建这个博客搜了好多教程，都不是特别全，就写了这篇教程

0. 前言

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
学习 Markdown 语法可以看 Github 官方文档：基本撰写和格式语法

1. 安装 Git 和 Node.js

  • Git
    从 Git官网 下载安装 Git，完成后用以下命令检查，有提示即安装正确
git --version

  • Node.js
    从nodejs官网 下载LTS 版本 Node.js并安装，完成后用以下命令检查，有提示即安装正确
node -v
npm -v

2. 安装 Hexo

新建一个文件夹，作为博客目录，cd 进入文件夹，运行

npm install hexo-cli -g

输入初始化部署命令：

hexo init

输入安装 Node.js 包管理器命令，安装所有的依赖：

npm install

输入生成本地预览命令：

hexo s

默认会生成 localhost:4000 端口的网址，在浏览器中可以访问。

hexo常用命令

hexo cl # 表示 clean，清理缓存
hexo g  # 表示 generate，生成HTML
hexo s  # 表示 server，启动服务
hexo d  # 表示 deploy，部署服务

3. 部署到GitHub

注册GitHub账户，过程不详细赘述。

生成SSH秘钥，用于推送代码。打开 gitbash, 输入命令

ssh-keygen -t rsa -C "username"   // username为你github的用户名

会在路径C:\Users\username\.ssh\下生成一个rsa文件，将文件内容添加到以下位置：

输入以下代码测试是否成功。

ssh -T git@github.com

创建一个新项目，用于保存和部署代码

配置_config.yml，注意修改时冒号后面一定要加空格。

#Site部分		站点描述自行修改
#URL部分		
    url: https://username.github.io	# 这个连接就是将来博客部署的地址
#Deployment部分
    type: git
    repo: https://github.com/username/username.github.io.git
    branch: master	# 根据自己建的项目更改

将代码推送到GitHub，运行以下命令

hexo cl
hexo g
hexo d

等待数分钟，打开 https://username.github.io 就能看到你的博客啦

4. 替换主题

首先在这里选择一个主题
上面的图片是预览页，下面的蓝色链接是 Github 项目页
选好主题后就要安装，在博客根目录下运行下面的命令将主题 Clone 到本地

git clone <theme-dir>.git themes\<theme-name>

其中为主题链接，为主题名称

比如我的主题是 ParticleX，就是

 git clone https://github.com/argvchs/hexo-theme-particlex.git themes\particlex

安装完成后，在博客根目录下的 _config.yml 中设置 theme 参数为你的主题名称，就可以切换主题。
