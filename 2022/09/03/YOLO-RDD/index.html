0.说明

本文是通过基于深度学习的目标检测技术来识别公路路面病害类别的一个完整实现。具体的说，从数据的采集到数据的标注，以及训练。通过改进的YOLOv5算法提升检测精度达到现有检测算法最好成绩，同时在Windows端，web端，Android端进行部署。想要了解更多详细信息，请参考这篇文章。

1.摘要

本文是通过基于深度学习的目标检测技术来识别公路路面病害类别的一个完整实现。

如需参考请引用以下文献：

本文是通过对YOLOv5算法改进的YOLO-RDD算法实现的一个路面病害检测系统。在自建数据集上取得第一名的成绩，比第二名YOLOv5x的MAP0.5高出**5.4%**同时检测速度有略微提升。并且实现了多端的部署，其中包括在Windows和web端的部署，安卓端部署正在进行中…

本代码库的文件结构说明：

    |-- back-end				后端推理服务
        |-- 					其它文件同上

    |-- front-end				前端页面展示
|-- images					说明文档插图

|-- readme.md					说明文档

3.代码运行

3.1 环境配置

本文中训练好的模型可以在以下链接 获取，提取码：yolo 。将下载的权重best.pt放在weights文件夹下即可。数据集由于合作公司不允许开源，因此无法提供。

a. win端部署

需要的软件：Pycharm、Anaconda（可选）

python环境：requirements.txt

环境搭建（没有接触过python的可参考这篇文章）：

1）首先安装Anaconda，并新建环境。

conda create -n yolo-rdd python=3.8.8
activate yolo-rdd

2）安装python依赖包

pip install -r requirements.txt

b. web端部署

需要的软件：Visual Studio Code、Pycharm、Anaconda（可选）

1）前端文件夹（front-end）下，先安装依赖：

npm install

2）后端（back-end）环境搭建同上Windows。

c. Android端部署

待更新。。。

3.2 运行

配置完环境后，通过以下代码进行对数据集训练：

python train.py --data data/crack.yaml --cfg yolo-rdd.yaml --weights '' --batch-size 8

其中，–data为数据集信息，–cfg为模型结构，–weights为预训练权重，–batch-size为batch-size大小，配置低的可以将其降低，最小为1。

通过以下代码进行测试：

python test.py

a. win端

可以直接使用如下代码运行，

$ python main.py

或使用如下代码编译为exe文件运行

$ pyinstaller -F -w main.py

编译为exe文件，可以直接双击运行。运行结果如下图：

同时，在前端文件夹（front-end）下运行以下代码：
